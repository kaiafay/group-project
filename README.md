# Let's Talk Recipes!

## Purpose
An application that allows users to search for recipes, and will read the recipes aloud via text-to-speech

## Built With
* HTML 
* Javascript
* jQuery
* ResponsiveVoice API
* TheMealDB API
* Bulma
